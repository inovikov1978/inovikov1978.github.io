License Setup

1. Generate a license file.
Go to the domain to license.
Run in console on website domain:
1. const FILE_NAME=`${btoa(window.location.host)}.json`;
2. FILE_NAME
Example file name:
3d3LmVsaXphYi5jb20=.json

File contents:
{ domain: "elizab.com" status: "active" }
2. Add File To License Server
Add file to: https://console.cloud.google.com/storage/browser/custom-product-builder-subscription-auth?project=buildateam-52&organizationId=56556588363
3. Make File Public (IMPORTANT)
Check 'Share Publicly' https://screencast.com/t/glAwEl1IJ
